### Apa Itu SI BABU?

**SI BABU** adalah **AI Agent modular berbasis Node.js** yang dirancang khusus untuk berjalan di terminal, terutama di **Termux (Android)** dan berbagai distribusi Linux.

Project ini dibuat dari nol sebagai proyek eksplorasi pribadi dengan tujuan menghadirkan pengalaman AI yang fleksibel, powerful, dan mudah digunakan langsung di terminal. SI BABU bukan sekadar chatbot biasa ia adalah agent cerdas yang mampu berpikir mandiri, menggunakan berbagai tools, mengingat konteks jangka panjang, melakukan riset, menulis kode, menganalisis error, serta bekerja secara kolaboratif melalui sistem **Multi-Agent Crew**.

### Fitur Lengkap

| Fitur | Deskripsi |
|-------|-----------|
| **Multi-Agent Crew System** | Terdiri dari Project Manager, Architect, Builder, Researcher, dan QA yang bisa berdiskusi, berdebat, saling memberikan feedback, serta berkolaborasi |
| **Natural Conversation** | Gaya ngobrol santai dan manusiawi menggunakan bahasa gaul Indonesia sehari-hari (gue/lo) |
| **Long-term Memory** | Mengingat percakapan lama dan belajar dari interaksi sebelumnya |
| **RAG** | Bisa membaca dan belajar dari dokumen serta file milik user |
| **Multi AI Provider** | Mendukung banyak provider (NVIDIA NIM, Groq, Gemini, OpenRouter, Ollama, dll) dengan fallback pintar |
| **Real-time Web Research** | Mencari dan merangkum informasi terkini dari internet |
| **Code Generation & Debugging** | Membuat kode lengkap, menganalisis error, dan memperbaiki secara otomatis |
| **Advanced Tool System** | Operasi file, eksekusi perintah, calculator, plugin system, dll |
| **Telegram Gateway** | Bisa dijalankan sebagai Telegram Bot |
| **Interactive Setup Wizard** | Setup awal yang ramah untuk pemula |
| **Plugin Architecture** | Mudah dikembangkan dengan menambahkan plugin baru |

### Instalasi

    git clone https://github.com/artone-arch/si-babu.git
    cd si-babu
    node setup.js

### Cara Menjalankan

    npm start          # Mode Chat Biasa
    npm run crew       # Mode Multi-Agent Crew

### English Version

SI BABU is a modular AI Agent built with Node.js, specifically designed to run in the terminal, especially on Termux (Android) and Linux.

Built from scratch as a personal exploration project, SI BABU aims to provide a flexible, powerful, and user-friendly AI experience directly in the terminal. It goes beyond a regular chatbot it can reason independently, use tools, retain long-term context, research, generate code, debug, and work collaboratively through a Multi-Agent Crew system.

**Installation**

    git clone https://github.com/artone-arch/si-babu.git
    cd si-babu
    node setup.js

**Running**

    npm start          # Normal Chat Mode
    npm run crew       # Multi-Agent Crew Mode
